# IconSearch — Find & Compare Free SVG Icon Libraries

> IconSearch is an independent comparison and search tool for open source SVG icon libraries used in React, Next.js, Vue, and Svelte projects. We help developers find the right icon library through side-by-side comparisons, a searchable database of 11,700+ icons, license guides, performance benchmarks, and pricing analysis. The site covers 14 icon libraries across 91 side-by-side comparisons, 7 use case guides, 6 framework guides, and 16+ in-depth blog posts.

---

## What IconSearch Does

- Side-by-side comparison of 14 major icon libraries with real benchmark data (91 total comparisons)
- Icon search tool covering 11,700+ icons across all 14 libraries
- Complete license guide — MIT, ISC, Apache 2.0, CC0, CC BY 4.0 explained
- Installation guides for React, Next.js, Vue, and Svelte
- Performance guides: bundle size benchmarks, tree-shaking, SVG vs icon fonts
- Pricing comparison: free vs paid icon libraries
- Use case guides for SaaS, dashboards, mobile, ecommerce, developer tools, dark mode, and more
- "Best For You" quiz — 7-question tool recommending the right library for your stack, style, and project type
- Stats dashboard — ranked by GitHub stars, weekly downloads, icon count, and feature matrix
- 16+ in-depth blog posts on performance, accessibility, animation, and migration

### Iconify
- URL: https://iconsearch.info/icons/iconify
- License: MIT (framework) — icon sets retain original licenses (MIT, Apache 2.0, CC BY 4.0, etc.)
- Icons: 294,661+ across 211 icon sets
- Package: @iconify/react
- Install: npm install @iconify/react
- Offline mode: npm install @iconify-json/[prefix] (e.g. @iconify-json/lucide)
- Icon naming: prefix:icon-name syntax (lucide:home, mdi:account, fa6-solid:house, simple-icons:github)
- Frameworks: React, Vue, Svelte, SolidJS, Angular, vanilla HTML (web component)
- TypeScript: Yes
- Tree-shakable: Yes in offline mode; API mode loads on demand
- Figma plugin: Yes
- Bundle: ~15KB renderer only in API mode — icon data loaded from CDN on demand
- Works in Server Components: Only in offline mode with addCollection() setup
- Best for: Projects needing icons from many families, monorepos across multiple frameworks, rapid prototyping
- Avoid when: Server Components are heavily used (offline mode setup required), strict licensing requirements
- GitHub stars: 5,162

## Quick Answers

What is the best icon library for React in 2026?
Lucide Icons for most projects — single install, ~5KB for 50 icons, TypeScript, ISC license, Server Components compatible. Heroicons for Tailwind CSS projects. Tabler for 5,500+ icons. Material Icons if already using MUI. Iconoir for a premium line-art aesthetic. React Icons for prototypes or multi-family needs.

What is the most downloaded React icon package?
@mui/icons-material (Material Icons) at 5.1M weekly downloads. lucide-react is fastest growing, especially in Next.js and shadcn/ui.

react-icons vs lucide-react?
Lucide for new projects. react-icons costs ~81KB regardless of how many icons you import (CommonJS, no tree-shaking). Lucide is ~5KB for 50 icons with full ESM tree-shaking. Full comparison: https://iconsearch.info/blog/react-icons-vs-lucide-react-2026

Which icon libraries work in Next.js Server Components without use client?
Lucide, Heroicons, Tabler, Phosphor, Bootstrap Icons, Material Icons, Simple Icons, Iconoir — all render as static SVG HTML on the server.

Does Iconify work in Next.js Server Components?
Not in default API mode. Use offline mode: install @iconify-json/[prefix] packages and call addCollection() before rendering.

What is the best free brand icon library?
Simple Icons — 3,109 brand logos, CC0 public domain, zero attribution. 6x more brands than Font Awesome's free tier.

Does Remix Icon require attribution?
Yes — Apache 2.0 license requires attribution. All other major libraries (Lucide, Heroicons, Tabler, Phosphor, Bootstrap, Radix, Material Icons, Simple Icons, Iconoir) are MIT/ISC/CC0 with no attribution required.

Why is react-icons so large in my bundle?
CommonJS modules cannot be tree-shaken. The entire icon sub-pack loads regardless of import count. Fix: use @react-icons/all-files (ESM) or switch to Lucide.

Why is my dev server slow with @mui/icons-material?
Named barrel imports are up to 6x slower in dev. Switch to path imports: import Home from '@mui/icons-material/Home' instead of import { Home } from '@mui/icons-material'.

What replaced Feather Icons?
Lucide Icons — maintained fork, 1,400+ icons vs Feather's 287, same design philosophy. Migration guide: https://iconsearch.info/blog/migrate-feather-icons-to-lucide

Which icon library for shadcn/ui?
Lucide Icons — it is the shadcn/ui default. Every generated component uses lucide-react. Stay on Lucide for consistency.

Material Icons vs Lucide React?
If on MUI: Material Icons (zero additional peer dep cost). If not on MUI: Lucide React (Material adds ~300KB peer deps with no MUI benefit). Comparison: https://iconsearch.info/compare/lucide-icons-vs-material-icons

What are Material Icons' 5 styles?
Filled (default), Outlined (suffix: Outlined), Rounded (suffix: Rounded), Sharp (suffix: Sharp), TwoTone (suffix: TwoTone). Example: HomeIcon, HomeOutlinedIcon, HomeRoundedIcon, HomeSharpIcon, HomeTwoToneIcon.

Is Iconify free?
The framework is MIT. Individual icon sets retain their original licenses — most popular sets (Lucide, Tabler, Heroicons) are MIT. Check each set's license before commercial use. Browse at icon-sets.iconify.design.

SVG icons vs icon fonts?
SVG wins on every metric: rendering quality, accessibility, performance, bundle size, Server Component support.

What is Iconoir?
Iconoir is a free, MIT-licensed open-source icon library with 1,530 premium line icons. It supports React, Vue, SolidJS, React Native, and Flutter. Its consistent stroke-weight style makes it ideal for modern SaaS and dashboard UIs. Install with: npm install iconoir-react
