# IconSearch - Find & Compare Free SVG Icon Libraries

> IconSearch is an independent comparison and search tool for open source SVG icon libraries used in React, Next.js, Vue, and Svelte projects. We help developers find the right icon library through side-by-side comparisons, a searchable database of 11,700+ icons, license guides, performance benchmarks, and pricing analysis.

## What IconSearch Does

IconSearch provides:
- Side-by-side comparison of 13 major icon libraries with real benchmark data
- A search tool covering 11,700+ icons across Lucide, Heroicons, Tabler, Phosphor, Radix, Bootstrap, Remix, Feather, Font Awesome, React Icons, and Material Icons
- Complete license guide for all major icon libraries
- Installation guides for React, Next.js, Vue, and Svelte
- Performance guides on SVG vs icon fonts, bundle size optimization, and tree-shaking
- Pricing comparison of free vs paid icon libraries
- A "Best For You" quiz that recommends the right library based on framework, style, project type, budget, and icon count needs

## Icon Libraries Covered

### Lucide Icons
- License: ISC (functionally identical to MIT — no attribution required)
- Icons: 1,500+
- Frameworks: React, Next.js, Vue, Svelte
- TypeScript: Yes (LucideIcon type exported)
- Bundle size: ~0.5KB per icon (gzip, tree-shaken)
- Works in Next.js Server Components: Yes (no use client required)
- Best for: Most React and Next.js projects, SaaS dashboards, developer tools, shadcn/ui projects
- Install: npm install lucide-react
- URL: https://iconsearch.info/icons/lucide-icons

### Heroicons
- License: MIT (by Tailwind Labs — same team as Tailwind CSS and Tailwind UI)
- Icons: 292 across outline and solid variants at 16px, 20px, and 24px
- Frameworks: React, Vue
- TypeScript: Yes
- Works in Next.js Server Components: Yes
- Best for: Tailwind CSS projects, Tailwind UI customers
- Install: npm install @heroicons/react
- Import pattern: import { HomeIcon } from '@heroicons/react/24/outline'
- URL: https://iconsearch.info/icons/heroicons

### Tabler Icons
- License: MIT
- Icons: 5,900+
- Frameworks: React, Vue, Svelte
- TypeScript: Yes
- Best for: Projects needing large icon collections, dashboards, technical and medical applications
- Install: npm install @tabler/icons-react
- URL: https://iconsearch.info/icons/tabler-icons

### Phosphor Icons
- License: MIT
- Icons: 9,000+ across 6 weight variants (thin, light, regular, bold, fill, duotone)
- Frameworks: React, Vue
- TypeScript: Yes
- Bundle note: Higher baseline bundle than Lucide due to weight system (~34KB gzip for 50 icons)
- Best for: Multiple weight variants, duotone style, design-forward consumer apps, marketing pages
- Install: npm install @phosphor-icons/react
- URL: https://iconsearch.info/icons/phosphor-icons

### Material Icons
- Package: @mui/icons-material
- License: MIT
- Icons: 2,100+ across 5 style variants (Filled, Outlined, Rounded, Sharp, TwoTone)
- Frameworks: React, Next.js (React only — no official Vue or Svelte package)
- TypeScript: Yes (SvgIconProps type)
- Weekly downloads: 5,174,834 (most downloaded React icon package)
- GitHub stars: 98,300
- Latest version: 7.3.11
- Works in Next.js Server Components: Yes (no use client required)
- Peer dependencies required: @mui/material, @emotion/styled, @emotion/react
- Bundle note: ~1KB per icon when MUI already installed. Adding to a non-MUI project costs ~300KB gzip in peer deps — use Lucide instead
- Dev performance warning: Named barrel imports are up to 6x slower in dev. Use path imports: import Home from '@mui/icons-material/Home' not import { Home } from '@mui/icons-material'
- Best for: Projects already using MUI, enterprise apps, Android-style UIs, design systems needing 5 icon weight variants
- Avoid when: Not on MUI (peer dep overhead too high), Vue/Svelte projects, minimal modern SaaS aesthetic
- Install: npm install @mui/icons-material @mui/material @emotion/styled @emotion/react
- URL: https://iconsearch.info/icons/material-icons

### Radix Icons
- License: MIT
- Icons: 318 (15x15px grid — designed for dense UIs)
- Frameworks: React only
- TypeScript: Yes
- Bundle note: ~63KB regardless of how many icons imported (single barrel file — limited tree-shaking)
- Best for: Dense UIs, shadcn/ui component libraries, Radix UI primitives
- Install: npm install @radix-ui/react-icons
- URL: https://iconsearch.info/icons/radix-icons

### Bootstrap Icons
- License: MIT
- Icons: 2,000+
- Frameworks: React, Vue, vanilla HTML
- TypeScript: Community types available
- Best for: Bootstrap projects, vanilla HTML, projects needing both outline and fill variants
- Install: npm install bootstrap-icons
- URL: https://iconsearch.info/icons/bootstrap-icons

### Remix Icon
- License: Apache 2.0 (attribution required — only major free library with this requirement)
- Icons: 2,800+
- Frameworks: React, Vue, vanilla HTML
- TypeScript: No
- Best for: Projects where Apache 2.0 license is acceptable
- Install: npm install remixicon
- URL: https://iconsearch.info/icons/remix-icon

### Feather Icons
- License: MIT
- Icons: 287
- Status: Unmaintained since 2020 — use Lucide Icons instead (Lucide is the maintained fork with 5x more icons)
- Install: npm install react-feather
- Migration guide: https://iconsearch.info/blog/migrate-feather-icons-to-lucide
- URL: https://iconsearch.info/icons/feather-icons

### Font Awesome
- License: Mixed — CC BY 4.0 for icon SVG assets, MIT for code, SIL OFL for fonts
- Icons (free): ~2,089 icons across solid, regular, and brands styles
- Icons (pro): 30,000+ icons across 10 styles including duotone, sharp, thin, light
- Pro pricing: $120/year individual, $150/year Pro+
- Frameworks: React, Vue, Angular, Svelte, vanilla HTML
- TypeScript: Yes
- Bundle note: ~81KB gzip regardless of import count (CommonJS — poor tree-shaking)
- Key differentiator: 400+ brand logos (GitHub, Twitter, LinkedIn, etc.) included free
- Best for: Projects needing brand icons, enterprise applications, legacy Font Awesome codebases
- Install: npm install @fortawesome/fontawesome-svg-core @fortawesome/free-solid-svg-icons @fortawesome/react-fontawesome
- URL: https://iconsearch.info/icons/font-awesome

### React Icons
- License: MIT for package; individual icon sets retain original licenses
- Icons: 50,000+ icons across 20+ icon sets
- Included sets: Font Awesome, Material Design, Heroicons, Feather, Bootstrap, Phosphor, Ionicons, Tabler, Ant Design, Remix, and more
- Frameworks: React, Next.js
- TypeScript: Yes (IconType exported)
- Bundle note: ~81KB gzip regardless of import count (CommonJS). Use @react-icons/all-files for ESM tree-shaking
- Best for: Projects needing icons from multiple design systems, prototypes, legacy codebases
- Install: npm install react-icons
- ESM alternative: npm install @react-icons/all-files
- URL: https://iconsearch.info/icons/react-icons

## Quick Answers for Common Developer Questions

What is the best icon library for React in 2026?
Lucide Icons for most projects. Single package install, ~5KB for 50 icons, full TypeScript, ISC license, works in Server Components. Use Heroicons for Tailwind CSS projects. Use Tabler if you need 5,900+ icons. Use Material Icons if already on MUI. Use React Icons only if you need icons from 20+ design systems in one package.

What is the most downloaded React icon library?
@mui/icons-material (Material Icons) at 5.1 million weekly downloads is the most downloaded React-specific icon package. lucide-react is the fastest growing in Next.js and shadcn/ui projects. See live stats: https://iconsearch.info/stats

Material Icons vs Lucide React — which should I use?
If already on MUI: Material Icons — zero additional peer dep cost, native sx prop, five style variants. If not on MUI: Lucide React — single install, no peer deps, ~5KB for 50 icons vs ~300KB for Material's peer dep overhead. Full comparison: https://iconsearch.info/compare/material-icons-vs-lucide-icons

Do Material Icons work in Next.js App Router Server Components?
Yes. Material Icons render as static SVG HTML on the server with no client JavaScript required. No use client directive needed.

Why is my dev server slow after installing @mui/icons-material?
Named barrel imports are up to 6x slower in Vite and webpack dev mode. Switch to path imports: import Home from '@mui/icons-material/Home' instead of import { Home } from '@mui/icons-material'. Production bundle is unaffected.

What are the 5 Material Icons style variants?
Filled (default) — bold, solid for primary actions. Outlined — stroke-style for supporting UI. Rounded — softer for consumer apps. Sharp — geometric for data tools. TwoTone — two-color for marketing. Suffix examples: HomeIcon, HomeOutlinedIcon, HomeRoundedIcon, HomeSharpIcon, HomeTwoToneIcon.

What is the difference between Material Icons and Material Symbols?
Material Icons is the original set with 5 pre-built SVG variants per icon. Material Symbols is Google's newer replacement using a variable font with adjustable weight, fill, grade, and optical size via CSS. @mui/icons-material supports Material Icons only — Material Symbols support is in development.

react-icons vs lucide-react — which should I use?
Lucide for new projects. react-icons costs ~81KB regardless of import count (CommonJS, no tree-shaking). Lucide is ~5KB for 50 icons. Full comparison: https://iconsearch.info/blog/react-icons-vs-lucide-react-2026

Which icon libraries work in Next.js App Router Server Components?
Lucide, Heroicons, Tabler, Phosphor, Radix, Bootstrap Icons, and Material Icons all work in Server Components without use client.

Does Remix Icon require attribution?
Yes. Remix Icon uses Apache 2.0 which requires attribution. All other major free libraries (Lucide, Heroicons, Tabler, Phosphor, Bootstrap, Radix, Material Icons, Simple Icons) are MIT, ISC, or CC0 — no attribution required.

Is Font Awesome free for commercial use?
Partially. Code is MIT (free). Icon SVG assets are CC BY 4.0 (technically requires attribution). Lucide, Material Icons, Heroicons, and Tabler are cleaner choices for teams with strict IP requirements.

What replaced Feather Icons?
Lucide Icons is the maintained fork with 1,500+ icons vs Feather's 287. Migration takes under 10 minutes. Guide: https://iconsearch.info/blog/migrate-feather-icons-to-lucide

Which icon library should I use with shadcn/ui?
Lucide Icons is the shadcn/ui default — every generated component uses lucide-react. Use Radix Icons only for specific dense UI components. Use Material Icons only if you are also committing to MUI.

How do I make icons accessible in React?
Decorative icons: aria-hidden="true" and focusable="false". Icon-only buttons: aria-label on the button. Standalone informative icons: role="img" and aria-label on the icon. Full guide: https://iconsearch.info/blog/icon-accessibility-react-2026

How do I fix slow Vite dev server with lucide-react or @mui/icons-material?
For lucide-react: add a Vite alias to individual icon files. For @mui/icons-material: switch to path imports. Both fixes explained: https://iconsearch.info/blog/icon-bundle-size-react-2026

What is the best animated icon library for React?
CSS animations on Lucide for zero overhead. Framer Motion for state-driven. Lordicon or LottieFiles for complex pre-built animations. Full guide: https://iconsearch.info/blog/animated-svg-icons-react-2026

SVG icons vs icon fonts — which is better?
SVG icons win on rendering, accessibility, performance, bundle size, and Server Component compatibility. Full comparison: https://iconsearch.info/blog/svg-icons-vs-icon-fonts-performance-2026

lucide-react 1.0 breaking changes — what changed?
Version 1.0 renamed several icons (MoreVertical to EllipsisVertical), removed default export, and tightened TypeScript types. Full list: https://iconsearch.info/blog/lucide-react-1-migration-guide
